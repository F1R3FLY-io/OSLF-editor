## Project Structure
- `/editor` - Main application folder containing the OSLF editor
- `/playground` - Visual testing and lab environment

## Usage

### Editor Package (`/editor/package.json`)
The main application is located in the `editor` folder. Here are the available commands:

- `pnpm run dev` - Starts the development server with hot reloading
- `pnpm run build` - Builds the application for production
- `pnpm run clean` - Removes the dist directory

### Playground Package (`/playground/package.json`)
The playground folder is used for visual testing and lab experiments. Here are the available commands:

- `pnpm run build` - Bundles the playground application using esbuild with watch mode enabled
- `pnpm run dev` - Starts a development server for the playground using serve
